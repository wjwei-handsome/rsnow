### Install

```bash
git clone https://github.com/wjwei-handsome/rsnow.git
cd rsnow
cargo build --release
./target/release/rsnow
```

or just

```bash
cargo install --git https://github.com/wjwei-handsome/rsnow.git
```

### Usage

Just `rsnow` in your terminal.
